# /memoriser — Mémorisation

1. Demander ce qu'il faut retenir :

```
Que mémoriser ?

A. Erreur à ne plus reproduire
B. Décision structurante
C. Préférence de génération
D. Autre
```

2. Reformuler la note en une entrée dense : `[YYYY-MM-DD] [Catégorie] — [note]`.
3. Consigner dans la mémoire agent (`~/.claude/agent-memory/`). Si la mémoire auto n'est pas activée : le signaler (`/config → Memory → Enable auto memory`), formuler la note quand même.
4. Confirmer en une ligne.
